# KhaymewNar

KhaymewNar is a mobile application developed as a university project using **Java (Android)**. The app allows users to explore available camps, view activities, manage reservations, and keep track of checklists for each camp. It also includes user authentication and profile management. This project is intended for educational purposes and is not deployed with real users.

## Features
- User authentication (Login and Sign Up)  
- User profile management  
- Browse available camps and their activities  
- Camp reservation system  
- Checklist for each camp to track progress  
- Community features (view posts, interact with other users)

---

## Technologies Used
- **Mobile Platform:** Android (Java)  
- **Database / Storage:** Local storage (in-app)  
- **UI:** XML layouts for Android  
- **Other:** RecyclerViews, Adapters, Intents
